Curso introdutório sobre Tecnologia da Informação, abordando conceitos fundamentais de hardware, software, sistemas, redes, dados e lógica computacional.

Este repositório utiliza:

- Markdown
- Git e GitHub
- Obsidian como base de organização do conhecimento
