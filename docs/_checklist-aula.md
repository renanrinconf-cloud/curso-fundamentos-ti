# Checklist de Qualidade — Aula

Antes de considerar uma aula concluída, verifique:

## Estrutura
- [ ] Título claro e numerado
- [ ] Contextualização presente
- [ ] Objetivos de aprendizagem definidos
- [ ] Conteúdo desenvolvido de forma progressiva
- [ ] Síntese ao final
- [ ] Navegação (aula anterior, próxima e índice)

## Clareza
- [ ] Linguagem adequada ao público iniciante
- [ ] Conceitos definidos antes de serem usados
- [ ] Exemplos quando necessário

## Consistência
- [ ] Segue o template padrão
- [ ] Numeração correta do arquivo
- [ ] Links funcionando no Obsidian

## Versionamento
- [ ] Commit com mensagem clara e pedagógica
