# Política de Qualidade Editorial

## Regra 1 — Aula só é considerada concluída se:

- Passar pelo checklist oficial
- Estiver versionada no Git
- Possuir navegação funcional

## Regra 2 — Toda aula nova deve:

- Ser criada em branch própria
- Ser integrada via Pull Request
- Ter commit(s) com padrão pedagógico

## Regra 3 — Revisões:

- Revisões de clareza incrementam versão patch
- Novas aulas incrementam versão minor

## Objetivo

Manter o curso consistente, evolutivo e auditável ao longo do tempo.
